Performs PIO read from the PCI BAR.

The specified address is treated as FIFO and will be read n times. The BAR will be automatically mapped and unmapped if necessary.

Parameters
[in,out]ctx- pcilib context
[in]bar- the BAR to read, use PCILIB_BAR_DETECT to detect bar by the specified physical address
[in]addr- absolute physical address to read or the offset in the specified bar
[in]access- the size of FIFO register in bytes (i.e. if access = 4, the 32-bit reads from FIFO will be performed)
[in]n- specifies how many times the data should be read from FIFO
[out]buf- the data will be placed in this buffer which should be at least n * access bytes long
Returns
- error code or 0 on success
